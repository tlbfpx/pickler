# Images Directory

This directory should contain the following images:

## Tab Bar Icons (81x81px recommended)
- home.png - Home icon (inactive)
- home-active.png - Home icon (active)
- ranking.png - Ranking icon (inactive)
- ranking-active.png - Ranking icon (active)
- me.png - Profile icon (inactive)
- me-active.png - Profile icon (active)

## Other Images
- default-avatar.png - Default user avatar (200x200px)
- default-event.png - Default event banner (800x400px)
- logo.png - App logo (200x200px)

## Note
You can create placeholder icons using WeChat Developer Tools or use SVG icons converted to PNG format.
